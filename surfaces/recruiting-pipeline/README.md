# Recruiting Pipeline Surface

This is the frontend surface for the Pravesh Pod recruiting pipeline.

**Live URL:** https://pravesh-recruiting-pipeline-co-pilot.apps.lemma.work/

## Views

- **All** — all candidates across all stages
- **Screening** — candidates being assessed (fit score not yet determined or < 60)
- **Interview Scheduled** — candidates who passed the JD score gate (≥ 60) and have slots sent

## Candidate Card

Each card shows:
- Candidate name + role applied for
- `fit` score badge (color-coded: green ≥ 80, yellow 60–79, red < 60)
- Pipeline status badge
- Source channel tag (EMAIL / WHATSAPP / REFERRAL)
- AI-generated screener rationale
- Contact details and years of experience
- Last activity and date

## Sort / Filter

- Sort by fit score (highest first) or most recent
- Filter by status tab

## Notes

The surface is auto-generated by Lemma from the `candidates` table schema
and configured via the pod's surface definition.
